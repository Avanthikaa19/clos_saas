Shortcut Action
Left Click Select an element in the viewport.
Delete Remove the selected element.
Position
Alt + A Toggle anchor mode for the selected element.
← & ↑ & → & ↓ Move the selected anchored element in all directions.
Size
Shift + ←&Shift + → Resize the width of the selected element (if it's supported).
Shift + ↑&Shift + ↓ Resize the height of the selected element (if it's supported).
Margins
Ctrl + ↑ & Ctrl + ↓ Increase the vertical margin of the selected element (if it's supported).
Ctrl + ← & Ctrl + → Increase the horizontal margin of the selected element (if it's supported).
Ctrl + Alt + ↑ & Ctrl + Alt + ↓ Decrease the vertical margin of the selected element (if it's supported).
Ctrl + Alt + ← & Ctrl + Alt + → Decrease the horizontal margin of the selected element (if it's supported).
